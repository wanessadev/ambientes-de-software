# ambientes-de-software
Este projeto contém a implementação de algoritmos fundamentais em JavaScript, desenvolvidos com foco em **Clean Code** e **Programação Defensiva**. A correção e robustez dos algoritmos são garantidas por testes unitários utilizando o framework **Jest**.

## Aluna
- **Nome**: Wanessa Honorato Vieira
- **Curso**: Análise e Desenvolvimento de Sistemas (ADS) - UNIFOR

## Algoritmos Implementados
- Contagem
- Fibonacci
- MDC (Máximo Divisor Comum)
- Número Primo
- Quicksort
- Somatório
